Web Development

I have little interest in web development, but I've dabbled in it at some points. I've got a CodePen page, mostly for the FreeCodeCamp projects.
My approach was to see the sample page and try to replicate them as faithfully as possible without looking at the code.
The most complete thing I've done was this ugly nutrition website. It is personally useful to compare nutrients, and especially protein profile for those who don't eat meat. It queries the FoodData Central and it then plots the various nutrients (summed into a recipe or against each other) using D3. It also supports importing and exporting recipes but I never really used that haha.
